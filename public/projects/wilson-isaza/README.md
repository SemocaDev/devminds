# Wilson Isaza - Imágenes del Proyecto

Coloca aquí las capturas de pantalla del sitio web de Wilson Isaza.

## Sistema dinámico:
Puedes agregar tantas imágenes como quieras. Solo actualiza el array `images` en `projects.json`.

## Cómo agregar imágenes:
1. Guarda tus capturas en esta carpeta (ej: `screenshot-1.jpg`, `screenshot-2.jpg`, etc.)
2. Actualiza el array `images` en `src/config/projects.json` para el proyecto `wilson-isaza`

## Ejemplo en projects.json:
```json
"images": [
  "/projects/wilson-isaza/screenshot-1.jpg",
  "/projects/wilson-isaza/screenshot-2.jpg"
]
```

## Recomendaciones:
- Formato: JPG o PNG
- Tamaño: 1920x1080 (16:9)
- Peso máximo: 500KB por imagen
- Contenido: Home, propuestas, secciones importantes
