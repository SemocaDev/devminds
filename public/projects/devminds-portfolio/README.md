# DevMinds Portfolio - Imágenes del Proyecto

Coloca aquí las capturas de pantalla del portfolio de DevMinds (este sitio).

## Sistema dinámico:
Puedes agregar tantas imágenes como quieras. Solo actualiza el array `images` en `projects.json`.

## Cómo agregar imágenes:
1. Guarda tus capturas en esta carpeta (ej: `screenshot-1.jpg`, `screenshot-2.jpg`, etc.)
2. Actualiza el array `images` en `src/config/projects.json` para el proyecto `devminds-portfolio`

## Ejemplo en projects.json:
```json
"images": [
  "/projects/devminds-portfolio/screenshot-1.jpg",
  "/projects/devminds-portfolio/screenshot-2.jpg"
]
```

## Recomendaciones:
- Formato: JPG o PNG
- Tamaño: 1920x1080 (16:9)
- Peso máximo: 500KB por imagen
- Contenido: Hero, proyectos, servicios, FAQ, contacto
