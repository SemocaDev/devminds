# Empo Guzmán - Imágenes del Proyecto

Coloca aquí las capturas de pantalla del sitio web de Empo Guzmán.

## Sistema dinámico:
Puedes agregar tantas imágenes como quieras. Solo actualiza el array `images` en `projects.json`.

## Cómo agregar imágenes:
1. Guarda tus capturas en esta carpeta (ej: `screenshot-1.jpg`, `screenshot-2.jpg`, etc.)
2. Actualiza el array `images` en `src/config/projects.json` para el proyecto `empo-guzman`

## Ejemplo en projects.json:
```json
"images": [
  "/projects/empo-guzman/screenshot-1.jpg",
  "/projects/empo-guzman/screenshot-2.jpg"
]
```

## Recomendaciones:
- Formato: JPG o PNG
- Tamaño: 1920x1080 (16:9)
- Peso máximo: 500KB por imagen
- Contenido: Home, servicios, información corporativa
