# Team Photos

Coloca aquí las fotos de los miembros del equipo.

## Nombres de archivo requeridos:

- `sebastian-morea.jpg` - Foto de Sebastian Morea Cañón
- `juan-gomez.jpg` - Foto de Juan David Gomez Perez

## Especificaciones técnicas:

- **Formato**: JPG o PNG
- **Tamaño recomendado**: 400x400px (cuadrado)
- **Peso máximo**: 200KB por imagen
- **Contenido**: Foto profesional o headshot

## Cómo agregar una foto:

1. Guarda la imagen en esta carpeta con el nombre correspondiente
2. El sistema automáticamente mostrará la foto en la página About
3. Si no hay foto, se mostrará un ícono de placeholder

## Notas:

- Las fotos deben ser cuadradas para mejor visualización
- Usa herramientas como TinyPNG para optimizar el tamaño
- Mantén un estilo consistente entre todas las fotos del equipo
